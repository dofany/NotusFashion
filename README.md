# NotusFashion

### IDE : SpringToolSuite4

### DB : oracle 11g

### Server : Apachi Tomcat v9.0

## NotusFashion 온라인 쇼핑몰 개인 프로젝트

- 사용자 로그인 및 회원가입 기능 추가
- 관리자 페이지 추가
- 고객센터 기능 
- 게시판 기능 추가
- 장바구니 기능 추가
- 관리자 배송 여부 기능 추가
- 회원정보 수정 기능 추가
- 회원탈퇴 기능 추가

### 마이바티스 연동시 mybatis-config 설정 후 root-context의 오라클 계정은 자기의 계정으로 변경
